### ClaudCodeを使ったブログアプリケーションチュートリアル

このリポジトリは、ClaudCodeを使用してブログアプリケーションを構築する方法を学ぶためのチュートリアルプロジェクトです。ClaudCodeは、AI駆動のコード生成ツールであり、開発者が効率的にコードを書くのを支援します。

### このリポジトリで学べること

- (1)シンプルにClaudeCodeを使ってのブログアプリケーションの構築方法
- (2)DDD（ドメイン駆動設計）の基本概念をClaudeCodeで実践する
- (3)`Custom Instructions`を活用してGitHub Copilotを使った開発でも効率化を図る
- (4)Context7、Serena、CipherなどのMCPサーバーを活用して、ClaudeCodeの能力を最大限に引き出す
